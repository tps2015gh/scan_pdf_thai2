🚀 How to Run AI PDF Scan

Follow these steps to get your local PDF analyzer up and running on your Optiplex 3070.

Prerequisite: Ensure you have Ollama installed and running.

Step 1: Install Models

1
Open your terminal (PowerShell or CMD) and run: 
Terminalollama pull qwen2.5-coder:7b
ollama pull nomic-embed-text

Step 2: Parse PDF Documents

2
Extract text from your PDF files into manageable chunks: 
Terminalpython parse_and_chunk.py

Step 3: Generate Embeddings

3
Create the vector database (Requires Ollama running): 
Terminalpython embed_local.py

Step 4: Run the Assistant

4
Start asking questions about your documents: 
Terminalpython main.py
Pro Tip: If you see character encoding issues, try running chcp 65001 in your terminal before running the scripts to enable UTF-8 support.
